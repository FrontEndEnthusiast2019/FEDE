A nunjucks, sass, js, and gulp based FEDE

A continuing project 14th april 2020

This project was made with simplicity in mind, as a starter kit for those wanting a basic nunjucks, sass, js, and gulp based frontend development environment.

Here it is, out of the box all files are compilable and renderable, placed within the public directory lies all of this code.

As a note do not edit your project from the public directory, only the app directory, keep to the projects structure with relative strictness and you should be fine.

If you know nunjucks, sass, and gulp you will be totally fine with this setup.

Current bugs include

  1. You must use the .nunjucks extension and not the .njk extension otherwise, nunjucks will not be able to render your html.
  2. Browser sync works through the CLI but to see the changes on the browser you need to restart gulp by typing "gulp" within your project directory each time (sadly).

100% now working, but still do not forget about the .nunjucks bug.
